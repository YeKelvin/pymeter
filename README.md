# Send-Anywhere
测试脚本执行引擎

## 项目结构
```
send-anywhere
    |-resources
        |-config.ini
	|-sendanywhere
        |-assertions
        |-configs
        |-controls
        |-engine
        |-functions
        |-listeners
        |-processors
        |-samplers
        |-testelement
        |-timers
        |-utils
```

## pipenv
项目使用pipenv管理依赖。在项目根目录下创建.venv目录后运行pipenv install即可。
