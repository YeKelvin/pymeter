# py-auto-test
测试脚本执行引擎。

## 项目结构
```
py-auto-test
    |-src
		|-core
			|-pers.kelvin.pyautotest
				|-assertion
				|-config
				|-engine
				|-exception
				|-function
				|-processor
				|-sample
				|-testelement
				|-thread
				|-util
```

## pipenv
项目使用pipenv管理依赖。在项目根目录下创建.venv目录后运行pipenv install即可。
